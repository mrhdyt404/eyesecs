Eye-Sec Site Checker

Analisis keamanan website secara real-time. Cek SSL, header keamanan, kerentanan, dan masalah keamanan lainnya dengan satu klik.

Pemindai Keamanan Situs Web

Memeriksa keamanan website. Harap tunggu...

Pemeriksaan SSL

Analisis sertifikat SSL untuk memastikan koneksi terenkripsi dan aman antara pengguna dan server.

Header Keamanan

Memeriksa header HTTP keamanan seperti CSP, HSTS, dan X-Frame-Options untuk mencegah serangan umum.

Deteksi Kerentanan

Mengidentifikasi kerentanan umum seperti XSS, SQL Injection, dan konfigurasi yang salah.

Alur Kerja Sistem

Lihat bagaimana Eye-Sec menganalisis keamanan website secara menyeluruh

1. Client

Pengguna mengirim permintaan analisis keamanan website melalui interface atau API

Permintaan diterima oleh endpoint /api/v1/url/check untuk diproses

3. Normalisasi URL

URL dinormalisasi dan diekstraksi domain utamanya untuk analisis konsisten

4. Google Safe Browsing

Pemeriksaan terhadap daftar website berbahaya dari database Google

Dokumentasi API

Cara Menggunakan API

Endpoint

POST /api/v1/url/check

Header

X-API-KEY: YOUR_API_KEY
Content-Type: application/json

Body Request

{
  "url": "https://example.com"
}

Contoh CURL

curl -X POST \
  -H "X-API-KEY: PUBLIC_GUEST_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://google.com"}' \
  https://eyesecs.site/api/v1/url/check
